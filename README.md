# Landing Page Project

## Overview
This project is part of [The Odin Project](https://www.theodinproject.com/) curriculum.  
The goal is to build a **responsive landing page** using **HTML** and **CSS**, focusing on clean structure, semantic markup, and styling fundamentals.

## Features
- Semantic HTML structure for accessibility and clarity
- CSS styling with external stylesheet
- Responsive design principles (flexbox/grid)
- Organized file structure for easy navigation

## Technologies Used
- HTML5
- CSS3

## Learning Objectives
- Practice writing semantic HTML
- Apply CSS styling and layout techniques
- Understand the workflow of building and documenting a project
- Strengthen Git/GitHub version control habits
